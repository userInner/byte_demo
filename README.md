# byte_demo V1 
## Running
## 请在已加入项目组环境下，执行git
````
mkdir byte_demo
cd byte_demo
git init
git checkout -b byte_demo.V1
git remote add origin git@github.com:userInner/byte_demo.git
git pull origin bytedemo.V1
````
